## Ukratko o projektu

Za lokalne tvrtke, fragmentirana digitalna prisutnost često umanjuje vrijednost brenda i ograničava privlačenje novih klijenata. Ovaj komercijalni projekt za **Adair Café Bar** imao je za cilj riješiti taj problem lansiranjem jedinstvene web platforme visokih performansi, popraćene potpunim vizualnim identitetom spremnim za produkciju.

Glavni izazov bio je pristupiti brendiranju i razvoju weba ne kao odvojenim zadacima, već kao integriranom ekosustavu od dizajna do programiranja. Prilagodio sam i opsežno modificirao responzivni osnovni framework kako bi odgovarao fizičkoj atmosferi kafića, osiguravajući da se teški resursi (poput prilagođenih cjenika visoke rezolucije i interaktivnih UI komponenti) učitavaju trenutačno i na mobilnim i na stolnim preglednicima korisnika.


## Moja uloga i izvedba
Preuzeo sam potpunu odgovornost za kreativno usmjeravanje i tehnički pipeline implementacije, pretvarajući dizajnerske resurse u web komponente spremne za produkciju.


## Tehnička implementacija i inženjerstvo
* **Responzivna arhitektura:** Prilagodio sam core CSS raspored koristeći Bootstrap i nativne medijske upite (*media queries*), stvarajući prilagodljivu mrežu (*grid*) koja zadržava savršeno poravnanje na mobilnim ekranima.
* **Interaktivne UI komponente:** Implementirao sam prilagođena JavaScript sučelja, jQuery interaktivne slojeve i glatke CSS animacije ključnih kadrova (*keyframe animations*), kao što su okretne kartice (*flip cards*) i interaktivni strukturni prijelazi, kako bih maksimizirao angažman korisnika.
* **Optimizacija resursa i infrastruktura:** Konfigurirao sam isporuku resursa produkcijske razine, minimizirajući veličinu slika, optimizirajući CSS rasporede i postavljajući sustav putem serverless statičnog hosting pipelinea koristeći GitHub Pages za visoku dostupnost bez operativnih troškova.


## Dizajnerski sustavi i vizualni identitet
* **Digitalni identitet brenda:** Izradio sam prilagođeni vektorski logotip, odabrao sustav tipografije i paletu boja prilagođenu tamnom načinu rada (*dark-mode*) koja odražava premium estetiku objekta.
* **Tiskani i digitalni materijali:** Dizajnirao sam službene strukturirane cjenike proizvoda i promotivne materijale koristeći specijalizirane pakete za digitalni dizajn, implementirajući stroge margine napusta za tisak (*print-bleed*) i prilagođena poravnanja stupaca za fizičku proizvodnju.


## Tehnički stack i matrica resursa
* **Frontend tehnologije:** HTML5, CSS3, JavaScript (ES6), Bootstrap 4, jQuery
* **Dizajn i grafički paketi:** Adobe Photoshop, Canva Core
* **Deployment i workflow:** Git kontrola verzija, GitHub Pages CDN infrastruktura
